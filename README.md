# Descripcion del proyecto

Programa desarrollado como proyecto final de la materia **Programacion 1**, que permite gestionar, consultar y analizar informacion sobre paises que existan dentro de un archivo .csv utilizando listas, diccionarios, funciones y tecnicas de filtrado, ordenamiento y estadisticas.

# Instrucciones de ejecución

**1**-Clonar el repositorio o descargar los archivos.

**2**-Asegurarse de que el archivo principal esté en la carpeta:

       └── TPI Programacion 1 - Gonzalo Cano - Nicolas Silvan
       
**3**-Ejecutarlo desde la consola con:
       
       └── python "TPI Programacio 1 - Gonzalo Cano - Nicolas SIlvan.py"

El sistema creará automáticamente el archivo "paises.csv" si no existe.
